BasketO




About


This e-commerce application offers a comprehensive solution for managing online sales with functionalities like user registration, product management, cart management, order processing, wishlist, and product ratings. Built using a robust technology stack, it ensures efficient data handling and security.



    Core Features


  • User Management: Robust user authentication and authorization using Spring Security.
  • Product Management: Manage product listings, categories, and ratings.
  • Cart and Wishlist: Seamless addition and management of products in the cart and wishlist.
  • Order Management: Secure order processing with integration of payment gateways.
  • Java Mail Sender: Email notifications for order confirmations and updates.


Technology Used


    Backend:
  • Spring Boot
  • MySQL Database
  • Spring Security
  • Java Mail Sender

    • Frontend:
  • Thymeleaf

    • Payment Gateways:
  • Razorpay
