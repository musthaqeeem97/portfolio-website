Ecolearn




About


The Ecolearn project at The Kyoto College of Graduate Studies for Informatics (KCGI) raises awareness about Sustainable Development Goals (SDGs) and equips students with practical tools and resources, like educational videos and an AI assistance bot.



    Core Features

    • Awareness Videos: Educational videos on SDGs, especially gender equality and quality education, embedded via iframes for seamless YouTube integration.
    • AI Assistance Bot: AI chatbot using JavaScript and OpenAI’s API, providing answers to IT and academic questions for both IT and non-IT students.
    • Essential Software Information: Curated list of software tools for KCGI students, with cards and links for easy access to downloads.
    • GitHub Guide: Introductory guide on GitHub usage, with links for account setup, repository navigation, and FOSS contributions.
    • User-Friendly Design: Responsive design using HTML and CSS, featuring cards, links, and embedded media for smooth navigation.
